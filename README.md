### Clone
```sh
git clon https://github.com/ShineShineDev/Vue3-Vuex4-Weather-App.git
```

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### USAGE
- API from openweathermap.org
- Vue 3
- Vue router 4
- Vuex 4 Module System & Map Helper
